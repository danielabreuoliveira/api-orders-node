## ⚙️ Instalação

Clone o repositório:

```bash

git clone https://github.com/seuusuario/seu-projeto.git

Entre na pasta do projeto:

cd seu-projeto

Instale as dependências:

npm install
▶️ Executando o projeto

Para iniciar o servidor:

npm start

ou

node server.js

Servidor rodará em:

http://localhost:3000
📡 Endpoints da API
Criar pedido
POST /orders

Exemplo de body:

{
  "numeroPedido": "12345",
  "valorTotal": 200.50
}
Listar pedidos
GET /orders
Buscar pedido por ID
GET /orders/:id
